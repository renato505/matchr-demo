# Logos das imobiliárias

Estes SVGs são placeholders neutros, criados apenas para o MVP não ficar com cards vazios.

Para usar os logos oficiais, substitua os arquivos mantendo exatamente estes nomes:

- `pilar.svg`
- `bossa-nova-sothebys.svg`
- `coelho-da-fonseca.svg`
- `local.svg`
- `axpe.svg`
- `cobogo.svg`
- `refugios.svg`
- `anglo.svg`
- `tamaras.svg`
- `jardins-co.svg`

Também funciona trocar por `.png`, mas nesse caso ajuste os caminhos no array `imobiliarias` dentro do `index.html`.
