# Matchr — Demo Vercel

Demo navegável do Matchr para corretores e investidores.

## Como publicar na Vercel

1. Crie um repositório novo no GitHub, por exemplo `matchr-demo`.
2. Faça upload de todos os arquivos desta pasta para o repositório.
3. Entre na Vercel e clique em **Add New > Project**.
4. Importe o repositório do GitHub.
5. Clique em **Deploy**.
6. Depois, em **Settings > Domains**, adicione `www.matchr.com.br`.

## Rodar localmente

```bash
npm install
npm run dev
```
